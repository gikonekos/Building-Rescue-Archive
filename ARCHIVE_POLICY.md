# Archive Policy

This repository preserves historical software for the
SHARP PC-E500 / PC-E550 pocket computer series.

The goal is **historical preservation and documentation**.

Principles of this archive:

- Preserve original source code and binaries
- Maintain historical accuracy
- Avoid modifying original programs
- Clearly separate reconstructed material from originals

If reconstruction or analysis is included,
it will be documented clearly and kept separate from original files.

This repository serves as a historical reference
for pocket computer software development.
