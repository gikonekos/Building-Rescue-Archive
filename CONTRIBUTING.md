# Contributing

This repository is a historical preservation archive.

The goal of this project is to preserve original software and documentation
for the SHARP PC-E500 / PC-E550 pocket computer series.

Contributions are welcome, especially:

- historical documentation
- magazine scans
- technical information
- original source material
- corrections to documentation

Important policy:

This repository focuses on **preservation of original materials**.
Please avoid submitting modern modifications of the software.

If you have historical materials related to this software,
please open an Issue or Pull Request.
